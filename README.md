## Install
```
npm install --save-dev eslint-config-gluons
```

## Usage
Add this to your [`.eslintrc.*`](http://eslint.org/docs/user-guide/configuring#configuration-file-formats) file:
```json
{
	"extends": "gluons"
}
```
